Assets Directory
================

This directory should contain the following files:

1. profile.jpg - Your professional profile photo
2. unc-logo.png - UNC Chapel Hill logo (provided)
3. databricks-cert.png - Databricks certification badge (provided)
4. aws-cert.png - AWS certification badge (provided)
5. oracle-cert.png - Oracle certification badge (provided)
6. resume.pdf - Your resume in PDF format

Please add these files to complete your portfolio website.

Image recommendations:
- Profile photo: 500x500px minimum, square format
- Certification badges: Use the official badges from your certifications
- UNC logo: Use the official university logo

The website will work without these images, but adding them will make it look more professional and complete.
